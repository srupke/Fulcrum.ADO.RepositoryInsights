# Privacy Policy

**Fulcrum Repository Activity**
**Publisher:** Scott Rupke
**Last updated:** June 11, 2026

## Overview

Fulcrum Repository Activity is an Azure DevOps extension that displays repository commit activity within your Azure DevOps organization. This policy describes how the extension handles data.

## Data Collection

This extension does **not** collect, store, transmit, or share any personal data or organizational data outside of your Azure DevOps environment.

All data accessed by this extension — including repository names, branch information, and commit metadata — is retrieved directly from your Azure DevOps organization via the Azure DevOps REST API and is displayed only within the extension's hub page.

## Data Usage

- Data is fetched at runtime in response to user interaction (selecting a date range and triggering a scan).
- No data is persisted by the extension beyond the current browser session.
- No analytics, telemetry, or tracking is performed by this extension.

## Third-Party Services

This extension does not integrate with or transmit data to any third-party services.

## Permissions

This extension requests the `vso.code` scope, which grants read-only access to source code, commits, and repository metadata within your Azure DevOps organization. No write access is requested or used.
